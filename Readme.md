# Quick & Dirty LED indicator

Install this:

https://github.com/tinue/APA102_Pi

and add the file in this repo.

Control examples


## Off

```
{
  "type": "off"
}
```


## Solid Color

```
{
  "type": "solid",
  "color": "0x0000ff"
}
```


## Ants

```
{
  "type": "ants",
  "color": "0x00ff00"
}
```


## Kitt
Larson scanner. Ignores color at the moment.
```
{
  "type": "kitt",
  "speed": "30"
}
```
